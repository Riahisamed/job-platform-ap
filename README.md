# 🚀 Job Platform API

## 📌 Description

Simple backend API built with NestJS where users can be created and stored in PostgreSQL.

## 🛠 Tech Stack

* NestJS
* PostgreSQL
* TypeORM

## ⚙️ Features

* Create user
* Get all users

## ▶️ Run project

```bash
npm install
npm run start:dev
```

## 📡 Endpoints

### POST /user

Create a new user

### GET /user

Get all users
